INSTALLATION — Pick the zip that matches your Blender version:

  • Blender 5.0 or later     →  mossy-fo4-blender-addon-v5.1.0-blender5x.zip
  • Blender 4.2 – 4.9        →  mossy-fo4-blender-addon-v5.1.0-blender42.zip
  • Blender 4.0 – 4.1        →  mossy-fo4-blender-addon-v5.1.0-blender4x.zip
  • Blender 3.6 LTS          →  mossy-fo4-blender-addon-v5.1.0-blender3x.zip

Check your Blender version:  Help → About Blender

Installing:
  1. Open Blender
  2. Edit → Preferences → Add-ons → Install
  3. Select the zip that matches your Blender version
  4. Enable the add-on (check the box)
